# Noé Mérida - Fotografía de Estudio

Sitio web profesional de fotografía de estudio en Acapulco.

## Características
- Portafolio de fotografía profesional
- Visualizador de imágenes con IA
- Análisis de estilo en vivo
- Estudio en vivo

## Tecnologías
- React
- Gemini AI API
- CSS

## Despliegue
Esta aplicación se aloja en Vercel.
